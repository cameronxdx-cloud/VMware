Software-defined networking that extends across data centers, clouds, and application frameworks. It supports multiple virtual platforms.

## Basic Concepts 
- Supports the virtualized network stack (ESXI, KVM, K8s) and bare metal 
- Separate configuration of the physical networking hardware 
- Use VMware Validated Design documentation 

## Physical Network
- The physical switches and routers that move data 
- Supports BGP
- Jumbo frames 
- STP portfast
